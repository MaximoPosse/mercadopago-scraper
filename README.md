# Scraper de promociones Mercado Pago

## Requisitos

- Node.js v16 o superior
- Puppeteer

## Instalación

```bash
cd mercadopago-scraper
npm install
```

## Ejecución

```bash
node index.js
```

## Salida

`mercadopago-scraper/data/promociones.json`

## Datos extraídos

- Comercio
- Beneficio
- Cuotas sin interés
- Imagen (URL completa)
- Descripción
- Vigencia / legales
- URL de la promoción

## Entrega

Para la entrega del trabajo, usar `mercadopago-scraper/README.txt`.
